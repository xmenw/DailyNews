# 每日新闻网站

## 介绍

+ 实现

  + 语言： PHP
  + 数据库： MySql
  + 服务器： Apache

+ 功能：

  + 前台管理

    1. 根据新闻分类查看新闻。

    2. 查看新闻详情。

    3. 查看热点新闻（阅读量前五）。

    4. 评论新闻。
    5. 登录。
    6. 注册。

  + 后台管理
    1. 上传新闻。
    2. 修改新闻。
    3. 删除新闻。
